# Code of Conduct

Zenicons aims to be a calm, constructive open-source project.

## Expected behavior

Please:

- be respectful in reviews and discussions,
- assume good intent before escalating,
- give actionable technical feedback,
- make room for different levels of experience.

## Unacceptable behavior

This project does not allow:

- harassment or personal attacks,
- discriminatory language or exclusionary behavior,
- doxxing, intimidation, or threats,
- deliberately disruptive spam in issues, discussions, or pull requests.

## Moderation

Project maintainers may edit, hide, lock, or remove content that harms the community or the project.

## Reporting

If you encounter conduct issues, open a private GitHub discussion or use the contact form at <https://bhaumicsingh.dev/>. Please include links or screenshots where possible.
